# Product Screenshots

This folder stores recruiter-facing screenshots for the GitHub README and portfolio launch materials.

Expected files:

- `landing.png`
- `login.png`
- `dashboard.png`
- `stock-research.png`
- `portfolio.png`
- `watchlist.png`
- `ask-ai.png`
- `admin.png`

To regenerate screenshots locally, start the backend, then run:

```powershell
cd C:\Users\shivk\fincredit-ai\frontend
npm run screenshots
```

Screenshots use local demo accounts only. Do not capture real credentials, private email, tokens, secrets, or production data.
